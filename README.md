# Personal Expense Tracker

## Overview

The Personal Expense Tracker is a Python application that helps users monitor their monthly expenses and savings.

The program tracks spending in the following categories:

* Food
* Transport
* Rent
* Entertainment

It then calculates total expenses, remaining balance, and savings percentage.

---

## Features

* Monthly income tracking
* Expense tracking
* Input validation
* Total expense calculation
* Remaining balance calculation
* Savings percentage calculation

---

## Categories

* Food
* Transport
* Rent
* Entertainment
